# Workspace Folder

Development workspace for design, testing, and code generation artifacts.

## Subfolders

- `design/` - Design documents
- `change/` - Change documentation
- `issues/` - Issue tracking
- `prompt/` - Code generation prompts
- `trace/` - Requirements traceability
- `audit/` - Compliance audits
- `test/` - Test documentation and results
- `ai/` - Temporary AI working materials (excluded from git)
